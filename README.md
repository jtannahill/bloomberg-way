## Quick Reference

### The Four-Paragraph Lead
```
P1 — Theme:   what happened + why it matters + how much + vs. what
P2 — Details: supporting facts, stats, named people, time/place
P3 — Quote:   authority with most at stake, plain English
P4 — Nut:     what's at stake for the reader, what happens next
```

### The Five Fs
```
Factual · First · Fastest · Final · Future
```

### Market Move Verbs
```
Up:   rose · gained · advanced · climbed · rallied · surged · jumped
Down: fell · dropped · declined · slumped · retreated · tumbled · lost
```

### Attribution Rules
```
✓ Use "said" — it's neutral
✗ Never: conceded, noted, claimed, explained, remarked, observed, opined
✓ Anonymous sources: "asked not to be named" + reason why
✗ Never direct quotes from anonymous sources
✗ Never: "sources said" — name the person
```

### Numbers
```
One–nine: spell out (unless before a unit or monetary amount)
10+: numerals
Percentages <10: one decimal place (9.7%, not 9.68%)
Percentages 10+: whole numbers (15%, not 15.3%)
Ranges: $10 million to $15 million (repeat the unit)
```

### Banned Words & Jargon
```
guidance · upside · downside · visibility · space · cautiously optimistic
restructuring · reorganization · strategic alternatives · shareholder value
profit warning · profit-taking · ongoing · upcoming · lavishly compensated
```
